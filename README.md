# Power BI Visual Calculations Exercise

## Overview
This project showcases my work completing the Microsoft Learn module:
**Create visual calculations in Power BI Desktop**.

The focus of this exercise is using visual-level DAX calculations such as:
- Profit calculations
- Versus Previous comparisons
- Running totals

## Skills Demonstrated
- Power BI Desktop
- Visual Calculations (Preview feature)
- DAX expressions
- Data modeling and aggregation
- Report design and interpretation
